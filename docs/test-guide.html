문제가 생길 때

앱 설치가 안 됨: 기존 앱 삭제 후 다시 설치합니다. APK를 카카오톡이 아닌 USB 또는 Google Drive로 옮깁니다. Play 프로텍트에서 세부정보 더보기 후 무시하고 설치를 선택합니다.

분석 중 멈춤: 같은 시간 seek 데드락 방지 패치가 들어가 있습니다. 그래도 멈추면 영상 길이와 파일 형식을 바꿔 테스트합니다. MP4 · 30초 내외 영상으로 먼저 확인합니다.

모델 준비 실패: 오프라인 자산이 포함되어 있는지 확인합니다. 아래 명령으로 자산 상태를 점검합니다.

npm.cmd run assets:check

개발자 검증 명령

npm.cmd run assets:check
npm.cmd run android:sync
cd android
$env:JAVA_HOME='C:\Program Files\Android\Android Studio\jbr'
.\gradlew.bat assembleDebug
