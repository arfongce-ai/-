문제 해결

설치가 안 됨: 기존 앱 삭제 후 USB 또는 Google Drive로 APK를 다시 옮겨 설치합니다.

분석 중 멈춤: 30초 내외 MP4 영상으로 먼저 테스트합니다.

모델 준비 실패: 오프라인 자산 체크 명령을 실행합니다.
