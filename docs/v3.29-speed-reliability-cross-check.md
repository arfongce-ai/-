# v3.29 분석 속도·정확성·신뢰성 교차검증

## 변경 사항

- v3.28의 안정적인 1패스 정밀 경계 스캔과 실제 `mediaTime` 동기화를 유지합니다.
- 경계 탐색에서 이미 검출한 포즈 프레임을 구간 평가 목표 시각에 맞춰 재사용합니다.
- 각 목표 시각에서 0.18초 이내의 관절 감지 프레임을 모두 고유하게 확보한 구간만 재사용합니다.
- 한 프레임이라도 누락되거나 시간 간격 기준을 넘으면 해당 구간 전체를 기존 정밀 seek·포즈 검출 방식으로 자동 폴백합니다.
- 대표 장면 스냅샷은 기존 정밀 seek 경로를 유지합니다.

## 속도 개선 검증

일반적인 품새 영상 조합을 합성해 기존 방식과 포즈 검출 호출 수를 비교했습니다.

| 영상 조건 | 결과 |
|---|---|
| 45초 / 18동작 | 구간 평가 프레임 전부 재사용 |
| 60초 / 18동작 | 구간 평가 프레임 전부 재사용 |
| 75초 / 25동작 | 구간 평가 프레임 전부 재사용 |
| 90초 / 30동작 | 구간 평가 프레임 전부 재사용 |
| 평균 예상 포즈 검출 감소 | 약 31.3% |

실제 기기 속도 개선 폭은 영상 코덱, 저장장치, 브라우저와 기기 성능에 따라 달라집니다.

## 정확성·신뢰성 보호

- 기존 구간 경계, 활성 구간 탐지, 동작명 매핑 로직은 변경하지 않았습니다.
- 요청한 구간 샘플 수 4~8개를 그대로 유지합니다.
- 재사용 시각 오차 상한을 0.18초로 제한합니다.
- 관절 미검출 합성 조건에서 정밀 폴백이 작동하는지 자동 검증합니다.
- 기존 엔진 테스트와 동작-구간 매핑 384회 시뮬레이션을 그대로 통과해야 합니다.

## 결과 기록

리포트 JSON의 `analysis_efficiency`에 다음을 저장합니다.

- 경계 탐색 포즈 검출 수
- 재사용한 포즈 프레임 수
- 정밀 폴백 포즈 검출 수
- 피한 중복 포즈 검출 수
- 실제 분석 소요시간
- 신뢰성 안전장치 설명
