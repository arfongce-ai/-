품새 AI 분석 v3.2

이번 버전 확인 포인트:
- 사용자가 제공한 품새 기술명 이미지 참고 반영 유지
- 대한태권도협회 경기규칙 페이지에서 2026년 품새 경기규칙 항목 존재 확인
- 앱은 공식 판정 앱이 아니므로 다음 표현 금지 검증 추가:
  감점 확정 / 공식 판정 / 합격 / 불합격 / 실격
- 유급자: 태극 1장~8장
- 유품자: 고려, 금강, 태백
- 유단자: 평원, 십진, 지태, 천권, 한수, 일여
- 품 수 데이터 검증 완료
- 모델 준비 버튼은 화면에서 숨김
- 분석 시작 옆에 재분석(초기화) 버튼 유지
- 결과지: 우선 확인할 부분 / 전체 부분 분리
- 각 구간 장면 사진 포함

주의:
PDF 전문 자동 추출 도구가 없어 PDF 본문 전체 대조는 하지 못했습니다. 공식 페이지 존재 확인 및 앱 데이터/표현 교차검증 기준으로 확인했습니다.
