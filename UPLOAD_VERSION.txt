태권도 품새 수련 훈련 v3.19

v3.19 핵심 변경
- 분석 시작 버튼을 화면 폭에 맞는 단일 주 버튼으로 정리했습니다.
- `훈련 분석 요약` 명칭을 `리포트`로 통일했습니다.
- 우선 확인 카드의 영상 캡처를 위쪽 중앙에 배치하고 피드백을 아래에 배치했습니다.
- 구간 감지를 초당 약 3회로 촘촘하게 수행하고 실제 움직임 최저점을 우선 선택합니다.

v3.18 핵심 변경
- 경기품새 피드백을 2026 경기규칙의 정확성·표현성 기준에 연결했습니다.
- 기술별로 서기, 막기, 지르기, 차기의 정확성 감점 위험 확인 항목을 표시합니다.
- 분석 전에 관절 움직임을 탐색해 구간 경계를 가까운 정지점으로 자동 보정합니다.
- 구간 감지 민감도를 보통, 높음(권장), 매우 높음으로 선택할 수 있습니다.
- 관절이 감지되지 않은 장면은 거짓 정지점 후보에서 제외합니다.

사용 흐름 및 UI 보완
- 선택한 품새와 영상 파일명에서 확인된 품새가 다르면 경고창을 표시합니다.
- 영상 내용 자체를 확정 판별하는 기능은 아니며 파일명에 품새명이 있을 때만 확인합니다.
- 재분석(초기화) 버튼은 화면에서 숨기고, 파일 또는 품새 변경 시 분석 결과를 자동으로 정리합니다.
- 구간 카드를 누르면 현재 위치를 유지한 채 팝업 영상으로 해당 구간을 재생합니다.
- 구간 카드의 `눌러 재생` 문구를 제거하고 기술명 색상을 밝게 보정했습니다.
- `결과지`를 `훈련 분석 요약`으로 변경했습니다.
- 훈련 분석 요약 상단에서 A4 한 장 크기의 JPG를 만들고 저장할 수 있습니다.

교차검증 보완 사항
- 정지 안정성은 마지막 구간의 잔여 움직임 속도와 속도 변화량을 함께 평가합니다.
- 일정한 속도로 계속 움직이는 장면이 정지 100점으로 판정되는 문제를 수정했습니다.
- 분석 표본 수를 일반적인 60~90초 영상에서 초당 약 1.5~2회 이하로 제한했습니다.
- 어깨와 골반 좌표의 신뢰도가 낮으면 신체 크기 정규화 계산에서 제외합니다.
- 현재 앱 문구와 자동 검증 문구를 일치시켰습니다.
- 정지 오판과 과도한 분석 표본 수의 재발을 막는 합성 엔진 테스트를 추가했습니다.

자동 검증 명령
- npm run assets:check
- npm run verify:cross
- npm run test:engine
- npm run test:all

주의 사항
- 이 앱은 영상 기반 훈련 보조 도구이며 공식 심사 또는 경기 판정을 대체하지 않습니다.
- 태극 1장은 상세 MVP 데이터입니다.
- 나머지 품새의 정확한 동작명과 지도 포인트는 지도자 검수 후 지속 보정이 필요합니다.
