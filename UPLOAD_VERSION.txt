품새 AI 분석 v3.4

이번 버전 확인 포인트:
- 깨진 한글 기술명 데이터 복구
- 태극 2장~8장, 고려~일여 기술명 count와 배열 길이 재검증
- 반복 피드백 원인 수정: 동작별 핵심 포인트 + 가장 약한 지표 1개만 표시
- 아래막기 / 몸통막기 / 손날 / 얼굴막기 / 바깥막기 / 안막기 규칙 분리
- 지르기 / 찌르기 / 치기 / 차기 / 옆차기 / 앞차기 규칙 분리
- 앞굽이 / 뒷굽이 / 주춤서기 / 범서기 / 학다리서기 / 꼬아서기 규칙 분리
- 독립 '초기화' 버튼 제거
- '재분석(초기화)' 버튼으로 분석 결과 초기화와 재분석 흐름 통합
- 훈련 화면의 'JPG 저장' 버튼 숨김
- '결과지 보기' 버튼은 결과지 전용 페이지로 이동
- 결과지 페이지 안에서만 'JPG 저장' 가능
- 사용자가 제공한 품새 기술명 이미지 참고 반영 유지
- 공식 판정처럼 보이는 표현 금지 검증 유지

검증 결과:
원본 프로젝트 cross-check 0 warning 통과.
오프라인 assets 검사 통과.
업로드 폴더 cross-check 0 warning 통과.
