품새 AI 분석 v3.9

이번 버전 변경 사항
- 기능을 바꾸지 않고 전체 화면 디자인을 세련되고 일관되게 정리했습니다.
- 차콜, 청록, 네이비 중심의 차분한 훈련 도구 색상 체계를 적용했습니다.
- 헤더, 버튼, 상태 지표, 결과 카드, 장면 카드의 간격과 시각적 우선순위를 통일했습니다.
- 심사 준비와 시합 준비 선택 영역을 더 쉽게 구분할 수 있게 개선했습니다.
- 모바일 하단 메뉴를 앱 형태로 다듬고 안전 영역 여백을 보강했습니다.
- 결과지의 평균 점수 표기와 심사·시합 모드별 피드백 구조는 기존 v3.8 동작을 유지합니다.

검증 결과
- 원본 프로젝트 cross-check: 0 warning
- 오프라인 필수 파일 검사: 통과
- Android 웹 자산 동기화: 완료
- 품새별 동작 수와 데이터 일관성 검사: 통과

사용 순서
1. 심사 준비 또는 시합 준비를 선택합니다.
2. 유급자, 유품자, 유단자 탭에서 품새를 선택합니다.
3. 영상을 선택합니다.
4. 분석 시작을 누릅니다.
5. 결과지에서 연습 포인트를 확인합니다.
