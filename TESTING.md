# 태권도 품새 수련 v3.21 테스트 가이드

## 1. 자동 검사

프로젝트 폴더에서 다음 명령을 실행합니다.

```text
npm run test:all
```

모든 항목이 `OK`로 끝나야 합니다.

## 2. PC 또는 Cloudflare Pages 검사

1. 수련품새 또는 경기품새를 선택합니다.
2. 유급자, 유품자, 유단자 탭에서 품새를 선택합니다.
3. 영상을 선택합니다. 품새와 영상은 어느 것을 먼저 선택해도 됩니다.
4. `분석 시작`을 누릅니다. 모델은 자동으로 준비됩니다.
5. 분석 중 영상 조작과 분석 버튼이 잠기는지 확인합니다.
6. 분석 완료 후 구간 카드 또는 사진을 누릅니다.
7. 영상이 선택한 구간으로 이동해 재생되는지 확인합니다.
8. 구간 카드를 누르면 현재 위치에서 팝업 재생창이 열리는지 확인합니다.
9. 하단 `리포트`를 눌러 별도 요약 페이지를 확인합니다.
10. `A4 요약 JPG 만들기`를 눌러 한 장짜리 JPG가 생성되는지 확인합니다.

## 구간 감지 민감도 검사

1. 먼저 `높음 · 권장`으로 같은 영상을 분석합니다.
2. 구간 카드의 시작·끝 장면이 기술 전환점과 맞는지 확인합니다.
3. 구간이 아직 앞뒤 동작과 섞이면 `매우 높음 · 정지점 우선`으로 변경해 다시 분석합니다.
4. 정지 없이 빠르게 이어지는 품새에서 구간이 지나치게 이동하면 `보통 · 경계 이동 적음`으로 낮춥니다.
5. 사람이 화면에서 벗어난 장면이 구간 경계로 선택되지 않는지 확인합니다.

민감도를 높이면 구간 경계 정확도가 좋아질 수 있지만 분석 시간이 조금 늘어납니다.

## 경기품새 규정 피드백 검사

1. `경기품새`를 선택하고 분석합니다.
2. 구간별 피드백에 해당 기술의 서기·막기·지르기·차기 정확성 확인 항목이 표시되는지 확인합니다.
3. 리포트에 정확성 및 표현성 기준이 표시되는지 확인합니다.
4. 앱이 감점을 확정하지 않고 지도자 확인이 필요하다고 안내하는지 확인합니다.

## 품새 선택 경고 검사

1. 품새에서 `태극 1장`을 선택합니다.
2. 파일명이 `태극2장-연습.mp4`인 영상을 선택합니다.
3. 선택 품새와 파일명이 다르다는 경고창이 나타나는지 확인합니다.

이 기능은 영상 동작을 판별하는 기능이 아닙니다. 파일명에 품새명이 적혀 있을 때만 잘못 선택할 가능성을 알려줍니다.

## 3. 데이터 일관성 검사

같은 영상, 같은 품새, 같은 훈련 모드로 다음 과정을 세 번 반복합니다.

1. 분석합니다.
2. 100점/평균과 우선 확인 구간을 기록합니다.
3. 화면을 이동한 뒤 다시 결과지를 확인합니다.

동일 세션에서는 결과가 바뀌지 않아야 합니다. 다른 영상 또는 품새를 선택하면 이전 화면 결과가 자동으로 정리됩니다.

## 4. 정지 안정성 검사

- 동작 끝에서 확실히 멈춘 영상은 정지 안정성이 높아야 합니다.
- 일정한 속도로 계속 움직이는 영상은 정지 안정성이 높게 나오면 안 됩니다.
- 동작 끝에서 흔들리는 영상은 우선 확인 구간에 표시되는지 확인합니다.

## 5. 모바일 실기기 검사

### Android

- Chrome에서 HTTPS 주소를 엽니다.
- 가로 영상에서 머리와 양발이 모두 보이는지 확인합니다.
- 60~90초 영상을 분석하며 발열, 멈춤, 앱 종료 여부를 확인합니다.
- 구간 카드 터치와 하단 메뉴가 한 번에 반응하는지 확인합니다.

### iPhone/iPad

- Safari에서 HTTPS 주소를 열고 홈 화면에 추가합니다.
- 영상 선택, 분석, 구간 재생, 결과지 이동을 확인합니다.
- 앱을 백그라운드로 보냈다가 돌아왔을 때 안내와 상태가 올바른지 확인합니다.

## 6. 문제 발생 시

- 분석 시작이 되지 않으면 품새와 영상이 모두 선택되었는지 확인합니다.
- 모델 로딩 실패 시 인터넷 연결 상태에서 한 번 열고, 오프라인 자산 검사를 실행합니다.
- 같은 영상 결과가 달라지면 품새와 훈련 모드가 같은지 확인합니다.
- 화면이 오래 멈추면 새로고침 후 짧은 영상으로 다시 검사합니다.
