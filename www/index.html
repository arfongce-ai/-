이 앱은 어떻게 쓰나요?

내가 품새 하는 모습을 영상으로 찍어서 넣으면, 동작을 하나하나 나눠서 "여기는 잘했어요", "여기는 이렇게 고치면 더 좋아요" 하고 알려주는 연습 도우미예요.

  1. ① 무엇을 볼지 골라요: '수련품새'(점수 없이 고칠 점만)와 '경기품새'(10점 점수까지) 중에 골라요.
  2. ② 품새를 골라요: 내 급수(유급자·유품자·유단자)를 누르고, 연습한 품새를 골라요.
  3. ③ 어떻게 찍었는지 골라요: 폰 몇 대로 찍었는지(정면 / 대각선 / 정면+측면 / 정면+측면+대각선)를 골라요.
  4. ④ 영상을 넣어요: 폰을 옆으로 눕혀서(가로) 머리부터 발끝까지 다 나오게 찍은 영상을 넣어요.
  5. ⑤ '분석 시작'을 눌러요: 다 될 때까지 화면을 끄지 말고 기다려요.
  6. ⑥ 결과를 봐요: 동작마다 영상을 다시 보고, 어디를 먼저 고치면 좋은지 확인해요.
  7. ⑦ 홈 화면에 추가하면 편해요: 안드로이드는 브라우저 메뉴에서 '앱 설치/홈 화면에 추가', 아이폰·아이패드는 공유 버튼에서 '홈 화면에 추가'를 눌러요.

잘 찍는 법: 폰은 가로로 · 온몸이 다 나오게 · 폰은 고정(흔들리지 않게) · 밝은 곳에서

정확한 동작이 궁금하면 국기원 공식 유튜브 품새 영상(34편)이나 세계태권도연맹(WT) 공식 영상, 국기원 태권도 교과서를 참고하세요. 이 앱의 평가 기준은 국기원·WT 공식 기준을 따릅니다.

이 앱은 혼자 품새 연습을 도와주는 참고용이에요. 선생님의 가르침이나 국기원 공식 심사를 대신하지는 않아요.
